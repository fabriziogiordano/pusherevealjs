## Usage

To controll the presentation you can type from your console:

```javascript
// Navigate Next page
$ ./controller next

// Navigate Prev page
$ ./controller prev

// Navigate Down page
$ ./controller down

// Navigate Up page
$ ./controller up

// Navigate Full presentation
$ ./controller full

// Return from Full presentation
$ ./controller nofull

// Navigate to page 3
$ ./controller 3
```

Some shortcuts:

```javascript
// Navigate Next page
$ ./controller n

// Navigate Prev page
$ ./controller p

// Navigate Down page
$ ./controller d

// Navigate Up page
$ ./controller u
```
